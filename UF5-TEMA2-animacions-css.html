Transicións CSS

Cambios suaves entre estados con transition, duración e timing-function.

🎨

Elevar

translateY + scale — 0.3s cubic-bezier

🌈

Cambio de cor

background + border-radius — 0.4s ease

🌀

Hue rotate

scale + filter — 0.3s ease

✨

Brillo

box-shadow + opacity — 0.3s ease

📄 Código CSS — Transicións
/* Transición básica */
.trans-card {
    transition: transform 0.3s cubic-bezier(0.34, 1.56, 0.64, 1),
                box-shadow 0.3s ease;
}
.trans-card:hover {
    transform: translateY(-8px) scale(1.02);
}

/* Cambio de cor con timing diferente */
.trans-card.color-shift {
    transition: background 0.4s ease, border-radius 0.4s ease;
}
.trans-card.color-shift:hover {
    background: var(--accent2);
    border-radius: 24px;
}

Animacións con @keyframes

Secuencias personalizadas con control de repetición, dirección e temporización.

Bounce
Pulse
Shake
Spin
Flip In
📄 Código CSS — @keyframes
/* Definición do keyframe */
@keyframes bounce {
    0%, 100% { transform: translateY(0); }
    40%      { transform: translateY(-30px); }
    60%      { transform: translateY(-10px); }
}

@keyframes pulse {
    0%, 100% { transform: scale(1); opacity: 1; }
    50%      { transform: scale(1.25); opacity: 0.7; }
}

/* Aplicación da animación */
.anim-bounce .anim-target {
    animation: bounce 0.8s ease infinite;
}
.anim-pulse .anim-target {
    animation: pulse 1.2s ease infinite;
}

Animacións de Entrada e Saída

Fade, Slide, Zoom — controlados con clases CSS e botóns.

Fade In
Slide Left
Zoom In
📄 Código CSS + JS — Entrada e Saída 
/* Fade In */
.entrance-box {
    opacity: 0;
    transform: translateY(30px);
    transition: opacity 0.5s ease, transform 0.5s ease;
}
.entrance-box.visible {
    opacity: 1;
    transform: translateY(0);
}

/* Slide Left */
.entrance-box.slide-left { transform: translateX(-40px); }
.entrance-box.slide-left.visible { transform: translateX(0); }

/* Zoom In */
.entrance-box.zoom-in { transform: scale(0.5); }
.entrance-box.zoom-in.visible { transform: scale(1); }

/* JavaScript para engadir/eliminar clase */
function showEntrance(type) {
    document.querySelectorAll('.entrance-box').forEach(el => {
        el.classList.toggle('visible', el.dataset.type === type);
    });
}

Animacións ao facer Scroll

Elementos que aparecen progresivamente usando Intersection Observer.

🚀

Deseño minimalista con animacións sutís.

🎯

Precisión

Interaccións pensadas para a experiencia de usuario.

⚡

Rendemento

Animacións optimizadas con transform e opacity.

🎨

Creatividade

Efectos visuais modernos sen sacrificar velocidade.

📄 Código JavaScript — Intersection Observer
// Intersection Observer — animacións ao facer scroll
const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            entry.target.classList.add('visible');
        }
    });
}, { threshold: 0.15 });

// Observa cada elemento con data-scroll
document.querySelectorAll('[data-scroll]')
    .forEach(item => observer.observe(item));

Efectos con Capas

Glassmorphism, overlays, blur e elementos flotantes.

Glassmorphism

backdrop-filter: blur(16px) sobre imaxe real con efecto vidro mate.

🌊

Overlay dinámico

Gradientes e capas superpostas con fotografía de fondo e transparencia.

💎

Elemento flotante

Animación float con delay escalonado sobre fondo de cristal/xeo.

📄 Código CSS — Glassmorphism / Capas
.glass-card {
    background-image: url('foto.jpg');
    background-size: cover;
    position: relative;
    overflow: hidden;
}

/* Capa de vidro mate (frosted glass) */
.glass-card::before {
    content: "";
    position: absolute;
    inset: 0;
    background: rgba(0,0,0,0.25);
    backdrop-filter: blur(16px);
    -webkit-backdrop-filter: blur(16px);
    z-index: 1;
}

/* Contido por riba da capa de blur */
.glass-content {
    position: relative;
    z-index: 2;
    color: #fff;
    text-shadow: 0 2px 12px rgba(0,0,0,0.6);
}

📄 Código HTML + JS — AOS
<!-- HTML: engadir atributos data-aos -->
<div data-aos="fade-up"
     data-aos-duration="800"
     data-aos-delay="100">
    Contido animado
</div>

<!-- JS: inicializar AOS -->
<script src="https://unpkg.com/aos@2.3.1/dist/aos.js"></script>
<script> AOS.init({ once: true }); </script>
