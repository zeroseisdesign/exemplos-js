# UF1306 – MÓDULO 2  
## UF2: Pruebas de funcionalidades y optimización de páginas web (90 horas)

---

# 1. ANÁLISIS Y ACTUALIZACIÓN DEL TEMARIO

## 1.1 Enfoque actualizado del módulo
- Validación de formularios moderna (HTML5 + JS + UX)
- Testing funcional en navegador real
- Optimización de rendimiento web
- Compatibilidad multi-dispositivo (responsive + cross-browser)
- Debugging profesional con DevTools
- Integración de componentes frontend actuales

---

## 1.2 Elementos obsoletos eliminados
- Frames → sustituidos por iframes modernos (YouTube, Maps, embeds)
- Efectos antiguos de HTML → sustituidos por CSS moderno
- Validación exclusivamente manual → sustituida por HTML5 + JS + UX
- Testing básico → ampliado a Lighthouse, Core Web Vitals y DevTools

---

## 1.3 Distribución temporal (90h)
- Tema 1: Validación de datos → 6–7,5 h  
- Tema 2: Efectos e interacción → 10–12 h  
- Tema 3: Testing y optimización → 10–12 h  
- Proyecto + examen → resto

Metodología: 80% práctica / 20% teoría

---

## 1.4 Estrategia didáctica (control de IA)
- Ejercicios en aula sin IA
- Debugging de código con errores intencionados
- Entregas con explicación obligatoria del proceso
- Uso de IA limitado a generación de prompts (no código final)
- Evaluación basada en razonamiento y corrección
