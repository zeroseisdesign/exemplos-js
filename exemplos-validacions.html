Exemplos de Validacións de Datos

Exemplos do Tema 1: Validación de datos en páxinas web con HTML5, JavaScript e CSS.

Función básica de validación

Función que verifica que un nome teña polo menos 3 caracteres:

function validarNome(nome) {
    return nome.length >= 3;
}

Validación con mensaxe de alerta

Se a validación falla, móstrase unha alerta:

if (!validarNome(nome)) {
    alert("O nome debe ter polo menos 3 caracteres");
}

HTML5 e JavaScript combinados

Validación de email con HTML5 e validity.typeMismatch:

<input type="email" id="correo" required>

const correo = document.getElementById("correo");
correo.addEventListener("input", () => {
    if (correo.validity.typeMismatch) {
        correo.setCustomValidity("Introduce un email válido");
    } else {
        correo.setCustomValidity("");
    }
});

Validación alfabética (JavaScript)

Función que permite só letras con expresión regular:

function validarTexto(texto) {
    return /^[A-Za-zÁÉÍÓÚáéíóúñÑ ]+$/.test(texto);
}

Validación alfabética (HTML5 con pattern)

Usando o atributo pattern de HTML5:

<input type="text"
       pattern="[A-Za-zÁÉÍÓÚáéíóúñÑ ]+"
       required>

Validación numérica (JavaScript)

Función que verifica se un valor é un número:

function validarNumero(numero) {
    return !isNaN(numero);
}

Validación numérica (HTML5)

Usando type="number" con min e max:

<input type="number" min="1" max="100">

Validación de data (JavaScript)

Función que comproba se unha data é válida con Date.parse():

function validarData(data) {
    return !isNaN(Date.parse(data));
}

Validación de data (HTML5)

Usando type="date":

<input type="date">

Restrinxir datas con min e max

Limitando o rango de datas con HTML5:

<input type="date" min="2025-01-01" max="2025-12-31">

Validación de contrasinal con HTML5

Contrasinal obrigatorio con mínimo 8 caracteres:

<input type="password" minlength="8" required>

Formulario con validación en submit

Validación ó enviar o formulario con preventDefault():

formulario.addEventListener("submit", (evento) => {
    const nome = document.getElementById("nome").value;
    if (nome.length < 3) {
        evento.preventDefault();
        alert("O nome debe ter mínimo 3 caracteres");
    }
});

Validación en tempo real co evento input

O campo valídase mentres o usuario escribe:

input.addEventListener("input", validarCampo);

function validarCampo() {
    if (input.value.length >= 3) {
        // Válido
    } else {
        // Inválido
    }
}

Formulario de rexistro completo

Formulario con usuario e email, validación ó enviar:

<form id="registro">
    <input type="text" id="usuario" required minlength="4">
    <input type="email" id="email" required>
    <button type="submit">Rexistrarse</button>
</form>

const formulario = document.getElementById("registro");
formulario.addEventListener("submit", (evento) => {
    const usuario = document.getElementById("usuario");
    if (usuario.value.length < 4) {
        evento.preventDefault();
        alert("O usuario debe ter mínimo 4 caracteres");
    }
});


CSS con pseudoclases :valid e :invalid

Os campos cambian de cor automaticamente segundo o seu estado:

input:valid {
    border: 2px solid green;
}
input:invalid {
    border: 2px solid red;
}
