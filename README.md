# Exemplos JavaScript

Este repositorio contén materiais didácticos e exemplos prácticos de JavaScript orientados á aprendizaxe dos fundamentos da linguaxe, o desenvolvemento de scripts e a manipulación de obxectos do navegador e do documento HTML.

## Contido do repositorio

- `index.html`: páxina inicial con exemplos básicos de operacións lóxicas, secuencias dun programa, pseudocódigo, entrada de datos, eventos e modificación dinámica de contido e estilos.
- `script.js`: código JavaScript asociado á páxina inicial. Inclúe comprobacións con formularios, uso de `prompt`, manipulación do DOM, eventos con `addEventListener` e cambios de estilo.
- `image.png`: imaxe empregada como ordinograma nun dos exemplos iniciais.
- `javascript-tema3.md`: apuntamentos do tema 3 sobre elementos básicos da linguaxe de guión: variables, tipos de datos, operadores, expresións, estruturas de control, funcións e instrucións de entrada e saída.
- `exemplos-javascript-tema3.html`: versión HTML con exemplos executables ou visualizables do tema 3.
- `javascrip-tema4.md`: apuntamentos do tema 4 sobre desenvolvemento de scripts, ferramentas de edición, recursos web, erros de sintaxe, erros de execución, mensaxes de erro e depuración no navegador.
- `exemplos-javascript-tema4.html`: versión HTML de referencia con exemplos relacionados coa depuración e o control de erros.
- `javascript-tema5.md`: apuntamentos do tema 5 sobre a xestión de obxectos da linguaxe de guión, incluíndo `window`, `document`, `navigator`, `location`, `history`, formularios e manipulación do DOM.
- `exemplos-javascript-tema5.html`: versión HTML con exemplos prácticos do tema 5 sobre obxectos do navegador, modificación de contido, estilos, atributos, creación e eliminación de elementos e traballo con formularios.

## Temas principais

O repositorio cobre estes bloques de aprendizaxe:

- Declaración e uso de variables en JavaScript.
- Tipos de datos: booleanos, numéricos, cadeas de texto, valores nulos e valores sen definir.
- Operadores de asignación, comparación, aritméticos, lóxicos, de bits e de cadeas.
- Estruturas de control como `if`, `while`, `for`, `break`, `continue` e `switch`.
- Creación e uso de funcións, parámetros, valores de retorno e funcións asociadas a eventos.
- Entrada e saída de datos mediante `prompt`, `alert`, `console.log`, `document.write` e elementos HTML.
- Ferramentas de desenvolvemento e depuración de JavaScript.
- Identificación e corrección de erros de sintaxe e de execución.
- Uso da consola do navegador e estruturas como `try`, `catch`, `finally` e `throw`.
- Acceso e manipulación dos obxectos do navegador e do documento.
- Selección, creación, modificación e eliminación de elementos do DOM.
- Manipulación de formularios, campos, atributos, clases e estilos CSS desde JavaScript.

## Como usar os exemplos

Pódense abrir directamente os ficheiros `.html` nun navegador web. Para unha experiencia máis completa, tamén se poden servir desde un servidor local como XAMPP, xa que o repositorio está preparado para funcionar dentro dun contorno web sinxelo.

Os ficheiros `.md` conteñen a explicación teórica e os exemplos de código correspondentes, mentres que os ficheiros `.html` permiten ver os exemplos estruturados como páxinas web.

## Obxectivo

O obxectivo do repositorio é servir como material de apoio para aprender JavaScript de maneira progresiva, combinando explicacións teóricas con exemplos prácticos que se poden revisar, modificar e executar no navegador.
