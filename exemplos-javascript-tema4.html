Referencia de JavaScript - Exemplos

Erros de sintaxe.

Prodúcense cando o código está mal escrito.

Exemplo:

let nome = "Ana

Falta pechar as comiñas.

Erros de execución

Prodúcense cando o código é correcto sintacticamente, pero falla ao executarse.

Exemplo:

let numero = 10;

console.log(numeroo);

A variable numeroo non existe.

Escritura do programa fonte

O programa fonte é o código escrito polo desenvolvedor.

Exemplo:

let idade = 20;

if (idade >= 18) {
  console.log("Maior de idade");
}

Compilación do programa fonte

JavaScript é interpretado.

Exemplo:

console.log("Programa iniciado");

TypeScript exemplo:

let idade: number = 25;

Corrección de erros de sintaxe

Exemplo incorrecto:

if (true {
  console.log("Erro");
}

Exemplo correcto:

if (true) {
  console.log("Correcto");
}

Pasos habituais:

  • Revisar a liña indicada pola consola.
  • Comprobar parénteses, chaves e comiñas.
  • Revisar puntos e comas ou nomes incorrectos.
  • Gardar e executar novamente o script.

Corrección de erros de execución

Exemplo:

let usuario = null;

console.log(usuario.nome);

Neste caso prodúcese un erro porque usuario non contén un obxecto.

Pasos habituais:

  • Abrir a consola do navegador.
  • Ler a mensaxe de erro completa.
  • Identificar a variable ou función que produce o problema.
  • Revisar os datos utilizados.
  • Corrixir o código e volver probalo.

Mensaxes de erro

As mensaxes de erro indican o tipo de problema e a liña onde se produce.

Exemplo habitual na consola:

Uncaught ReferenceError: variable is not defined

As mensaxes de erro adoitan indicar:

  • O tipo de erro.
  • O arquivo onde se produce.
  • A liña do código afectada.
  • A descrición do problema.

Funcións para controlar os erros

Instrución Descrición
try Executa un bloque de código
catch Captura o erro producido
finally Executa código ao finalizar
throw Xera un erro manualmente

Exemplo:

try {
  let resultado = numero + 10;
} catch (erro) {
  console.log("Produciuse un erro");
}

Uso básico da consola

A consola permite mostrar información, detectar erros, executar código manualmente, comprobar valores de variables.

Exemplo:

let nome = "Ana";

console.log(nome);

Exercicio práctico

Crear unha páxina web que declare varias variables, realice unha operación matemática, mostre datos por consola, solicite un nome mediante prompt(), mostre unha mensaxe personalizada, inclúa un erro intencionado, localice e corrixa o erro usando a consola do navegador.

Exemplo inicial:

let nome = prompt("Introduce o teu nome");

console.log("Benvido " + nome);

let numero1 = 10;
let numero2 = 5;

console.log(numero1 + numero2);
